# JavaScript Mastery Exercises

Complete exercises to master JavaScript fundamentals including loops, array methods, and functions.

## Exercises

### 1. Loop Mastery
📁 [01-loop-mastery.js](./01-loop-mastery.js)
- Multiplication tables using nested loops
- While loop patterns
- Different iteration techniques

### 2. Array Methods Practice
📁 [02-array-methods.js](./02-array-methods.js)
- Work with student dataset
- Practice filter, map, reduce, sort, some, every
- Data transformation and analysis

### 3. Function Challenges
📁 [03-function-challenges.js](./03-function-challenges.js)
- Implement isPalindrome function
- String manipulation
- Function creation patterns

## How to Run

### Node.js
```bash
node 01-loop-mastery.js
node 02-array-methods.js
node 03-function-challenges.js
```

### Browser Console
Copy and paste the code into your browser's developer console.
