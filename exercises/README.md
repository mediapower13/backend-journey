# Coding Exercises

Practice exercises for backend development concepts.

## Topics

### 1. Data Structures & Algorithms
- Arrays and strings
- Linked lists
- Trees and graphs
- Hash tables
- Sorting and searching

### 2. API Design
- RESTful conventions
- Error handling
- Status codes
- Request validation

### 3. Database Queries
- SQL practice
- Query optimization
- Indexes
- Relationships

### 4. Authentication
- JWT implementation
- Session management
- OAuth flows
- Password hashing

## Exercise Format
Each exercise should include:
- Problem description
- Requirements
- Test cases
- Solution
- Learnings
